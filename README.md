# CareerCompass AI

## Smart Career Guidance System

CareerCompass AI is a smart web-based career guidance application developed to help students and freshers identify suitable career paths based on their interests, logical reasoning, aptitude, and technical skills.

The system provides career recommendations, personalized learning roadmaps, and industry-relevant skill suggestions using a user-friendly interface.

---

# Features

- User Registration & Login
- Career Assessment Quiz
- AI-Based Career Recommendation
- Personalized Dashboard
- Learning Roadmap
- Trending Technologies Section
- Salary Insights
- Career Progress Tracking

# Modules

## Recommendation Module

## Dashboard Module
Displays learning progress, insights, salary information, and trending technologies.
